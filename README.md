# Tesla-landing-page
A replica of the Tesla landing page

## Description
This project is a static HTML, CSS, and JavaScript replica of the Tesla website's landing page. It features a responsive design mimicking Tesla's layout, including navigation, hero section, model showcases, accessories, and footer.

## Features
- Responsive navigation bar with Tesla logo and menu links
- Hero section with call-to-action buttons
- Sections for different Tesla models (Model S, Model 3, Model X, Model Y)
- Accessories section with grid layout
- Footer with links and copyright information
- Smooth scrolling navigation
- Mobile-friendly responsive design
- Clean, modern styling inspired by Tesla's design

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (ES6)

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/Ntsika-Ngilane-dev/Tesla-landing-page.git
   ```
2. Navigate to the project directory:
   ```
   cd Tesla-landing-page
   ```

## Usage
Open `index.html` in your web browser to view the landing page. No additional setup or server is required as it's a static website.
